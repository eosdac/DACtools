# Purpose
Vote gathering tool to discover voting statistics for the top EOS accounts

# Caution
The top500.csv is a predetermined list which you can change.

# Tech Details
These are the details provided by the resulting csv:
- account
- proxy
- staked
- last_vote_weight
- proxied_vote_weight
- is_proxy
- votes
